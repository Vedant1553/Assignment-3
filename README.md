# TaskFlow API

## Description

TaskFlow API is a RESTful backend service built using Node.js and Express.

It handles CRUD operations for task management.

---

## Features

- Create Tasks
- Read Tasks
- Update Tasks
- Delete Tasks
- REST API
- Middleware Support
- Error Handling
- CORS Enabled
- Security Headers

---

## Technologies Used

- Node.js
- Express.js
- CORS
- Morgan
- Helmet

---

## Installation

```bash
npm install
```

---

## Run Server

```bash
npm start
```

---

## API Endpoints

| Method | Endpoint | Description |
|---|---|---|
| GET | /api/tasks | Get all tasks |
| GET | /api/tasks/:id | Get task by id |
| POST | /api/tasks | Create task |
| PUT | /api/tasks/:id | Update task |
| DELETE | /api/tasks/:id | Delete task |
